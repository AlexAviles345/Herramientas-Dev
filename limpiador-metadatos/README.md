# 🧼 Herramientas de Limpieza de Metadatos

Esta categoría agrupa diferentes herramientas diseñadas para eliminar metadatos, rastros y huellas digitales ocultas en archivos. La privacidad es primordial al compartir archivos en la red pública.

## 📋 Índice de Limpiadores

| Herramienta | Descripción | Tecnologías |
| :--- | :--- | :--- |
| **[Limpiador de Imágenes](./limpiador-metadatos-imagenes)** | Elimina metadatos EXIF (coordenadas GPS, dispositivo, fechas) de imágenes redibujando los píxeles a través de Canvas API. Todo ocurre en tu navegador (100% privado, sin backend). | HTML, CSS, JS Vanilla (Canvas) |

> *Próximamente se agregarán limpiadores para otros tipos de archivos (PDFs, documentos, etc).*
